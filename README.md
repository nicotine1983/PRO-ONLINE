# PRO-ONLINE

Prototype d’application dans le domaine du **levage, de la manutention et du transport**.

> **Attention :** le dépôt est actuellement un prototype. Les données affichées sont des exemples et ne doivent pas être utilisées pour décider d’un levage, d’un calage, d’un élingage, d’une capacité ou d’un transport réel.

## État actuel

- API Node.js / Express : `src/main.js`
- Cockpit web statique : `cockpit-web/`
- Données de démonstration en mémoire
- Aucun calcul métier critique validé

Le contexte technique vérifié et les limites connues sont détaillés dans [`docs/PROJECT_CONTEXT.md`](docs/PROJECT_CONTEXT.md).

## Travail avec ChatGPT et Codex

Le dépôt sert de mémoire commune :

1. lire [`AGENTS.md`](AGENTS.md) ;
2. consulter la tâche active dans [`docs/CURRENT_TASK.md`](docs/CURRENT_TASK.md) ;
3. respecter les décisions de [`docs/DECISIONS.md`](docs/DECISIONS.md) ;
4. consigner les résultats dans [`docs/CHANGELOG_AGENT.md`](docs/CHANGELOG_AGENT.md).

La procédure complète est décrite dans [`docs/WORKFLOW_CHATGPT_CODEX.md`](docs/WORKFLOW_CHATGPT_CODEX.md).
